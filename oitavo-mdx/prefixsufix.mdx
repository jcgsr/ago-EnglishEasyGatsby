Prefixos são letras que são adicionadas no **início** de uma palavra para formar uma nova palavra com um _significado diferente_. Os prefixos podem, por exemplo, criar uma nova palavra de significado oposto ao da palavra à qual o prefixo está anexado. Eles também podem tornar uma palavra negativa ou expressar relações de tempo, lugar ou maneira.

Sufixos, por sua vez, são letras ou grupos de letras adicionadas no **final** de uma palavra formando uma nova, sendo essa nova palavra, na maioria das vezes, de uma classe gramatical diferente. Vê as tabelas abaixo para consulta dos prefixos e sufixos mais comuns.

## Prefixos

| Prefixo           | Significado (Exemplos)     |
| ----------------- | -------------------------- |
| anti-             | oposto a (anti-government) |
| auto-             | próprio (autobiography)    |
| de-               | reverter (demotivate)      |
| dis-              | reverter (disagree)        |
| il-, im-, in-, ir | não (illegal)              |
| mid-              | meio midday()              |
| mis-              | mal (misunderstanding)     |
| over-             | demais (overcook)          |
| post-             | depois (post-war)          |
| pre-              | pré (pre-war)              |
| pro-              | a favor de (pro-communist) |
| re-               | de novo (reconsider)       |
| semi-             | metade (semicircle)        |
| un-               | não (unhappy)              |

**Obs.:** Não há absolutamente nenhuma regra sobre usar ou não usar hífen.

## Sufixos

### Substantivos

| Sufixo       | Significado (Exemplos)            |
| ------------ | --------------------------------- |
| -acy         | estado ou qualidade (privacy)     |
| -al          | ato ou processo de (refusal)      |
| -ance, -ence | estado ou qualidade (maintenance) |
| -dom         | estado de estar/ser (freedom)     |
| -er, -or     | quem faz (trainer)                |
| -ism         | doutrina (skepticism)             |
| -ist         | alguém que (plagiarist)           |
| -ity, -ty    | qualidade de (serenity)           |
| -ment        | condição de ( punishment)         |
| -ness        | estado de ser/estar (sadness)     |
| -ship        | posição mantida (fellowship)      |
| -sion, -tion | estado de ser/estar (transition)  |

### Verbos

| Sufixo     | Significado (Exemplos) |
| ---------- | ---------------------- |
| -ate       | tornar-se (regulate)   |

| -ify, -fy  | fazer (terrify)        |
| -ize, -ise | tornar-se (socialize)  |

### Adjetivos

| Sufixo       | Significado (Exemplos)        |
| ------------ | ----------------------------- |

| -esque       | remanescente de ( burlesque)  |
| -ful         | cheio de (resentful)          |
| -ic, -ical   | pertencente (musical)         |
| -ious, -ous  | caracterizado por (studious)  |
| -ish         | ter a qualidade de (blackish) |
| -ive         | ter a natureza de (creative)  |
| -less        | sem (endless)                 |
| -y           | caracterizado por (lazy)      |
