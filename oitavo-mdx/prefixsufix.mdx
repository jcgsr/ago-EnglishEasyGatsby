Prefixos são letras que são adicionadas no **início** de uma palavra para formar uma nova palavra com um _significado diferente_. Os prefixos podem, por exemplo, criar uma nova palavra de significado oposto ao da palavra à qual o prefixo está anexado. Eles também podem tornar uma palavra negativa ou expressar relações de tempo, lugar ou maneira.

Sufixos, por sua vez, são letras ou grupos de letras adicionadas no **final** de uma palavra formando uma nova, sendo essa nova palavra, na maioria das vezes, de uma classe gramatical diferente. Vê as tabelas abaixo para consulta dos prefixos e sufixos mais comuns.

## Prefixos

| Prefixo           | Significado            | Exemplos                                                                              |
| ----------------- | ---------------------- | ------------------------------------------------------------------------------------- |
| anti-             | contra/oposto a        | anti-government (antigovernamental)                                                   |
| auto-             | próprio                | autobiography (autobiografia)                                                         |
| de-               | reverter ou mudar      | demotivate (desmotivar)                                                               |
| dis-              | reverter ou mudar      | disagree (discordar)                                                                  |
| il-, im-, in-, ir | não                    | illegal (ilegal), impossible (impossível), insecure (inseguro), irregular (irregular) |
| mid-              | meio                   | midday (meio-dia)                                                                     |
| mis-              | incorretamente, mal    | misunderstanding (mal-entendido)                                                      |
| over-             | demais                 | overcook (cozinhar demais)                                                            |
| post-             | depois                 | post-war (pós-guerra)                                                                 |
| pre-              | pré, anterior          | pre-war (pré-guerra)                                                                  |
| pro-              | a favor de             | pro-communist (pró-comunista)                                                         |
| re-               | de novo                | reconsider (reconsiderar)                                                             |
| semi-             | metade                 | semicircle (semicírculo)                                                              |
| un-               | remover, inverter, não | unhappy (infeliz)                                                                     |

**Obs.:** Não há absolutamente nenhuma regra sobre usar ou não usar hífen.

## Sufixos

### Substantivos

| Sufixo       | Significado                  | Exemplos                 |
| ------------ | ---------------------------- | ------------------------ |
| -acy         | estado ou qualidade          | privacy (privacidade)    |
| -al          | ato ou processo de           | refusal (recusa)         |
| -ance, -ence | estado ou qualidade          | maintenance (manutenção) |
| -dom         | lugar ou estado de estar/ser | freedom (liberdade)      |
| -er, -or     | quem faz (a ação do verbo)   | trainer (treinador)      |
| -ism         | doutrina, crença             | skepticism (ceticismo)   |
| -ist         | alguém que                   | plagiarist (plagiador)   |
| -ity, -ty    | qualidade de                 | serenity (serenidade)    |
| -ment        | condição de                  | punishment (pinição)     |
| -ness        | estado de ser/estar          | sadness (tristeza)       |
| -ship        | posição mantida              | fellowship (camaradagem) |
| -sion, -tion | estado de ser/estar          | transition (tansição)    |

### Verbos

| Sufixo     | Significado        | Exemplos               |
| ---------- | ------------------ | ---------------------- |
| -ate       | tornar-se          | regulate (regular)     |
| -en        | tornar-se          | enlighten (iluminar)   |
| -ify, -fy  | fazer ou se tornar | terrify (aterrorizar)  |
| -ize, -ise | tornar-se          | socialize (socializar) |

### Adjetivos

| Sufixo       | Significado            | Exemplo                                         |
| ------------ | ---------------------- | ----------------------------------------------- |
| -able, -ible | capaz de ser/estar     | edible (comestível)                             |
| -al          | pertencente            | regional (regional)                             |
| -esque       | remanescente de        | burlesque (burlesco)                            |
| -ful         | notável por (cheio de) | resentful (ressentido - cheio de ressentimento) |
| -ic, -ical   | pertencente            | musical (musical)                               |
| -ious, -ous  | caracterizado por      | studious (estudioso)                            |
| -ish         | ter a qualidade de     | blackish (enegrecido)                           |
| -ive         | ter a natureza de      | creative (criativo)                             |
| -less        | sem                    | endless (sem fim, infinito)                     |
| -y           | caracterizado por      | lazy (preguiçoso)                               |
