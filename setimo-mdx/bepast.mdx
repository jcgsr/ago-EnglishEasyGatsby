O Passado Simples do verbo **To Be** tem duas formas quando conjugado: **was**, usado com as pessoas _I, he, she, it_ e **were**, usado com o restante _you, we, they_. A tradução dessas duas formas, como de constume, depende das pessoas em questão.

Presta atenção que onde houver as traduções **era, eram, éramos**, pode-se perfeitamente trocá-las por **estava, estavam, estávamos**. Outra coisa: não existe contração das formas **was, were** na afirmativa como existe no presente (_I'm, you're, he's_, etc). Na negativa, pode ser tanto **was not, were not** quanto _wasn't, weren't'_. Confere as tabelas abaixo para a conjugação completa.

## Conjugação

| Afirmativa | be (simple past) |
| ---------- | ---------------- |
| I was      | Eu era           |
| You were   | Você era         |
| He was     | Ele era          |
| She was    | Ela era          |
| It was     | Ele/ela era      |
| We were    | Nós éramos       |
| You were   | Vocês eram       |
| They were  | Eles/elas eram   |

| Negativa     | be (simple past)   |
| ------------ | ------------------ |
| I wasn't     | Eu não era         |
| You weren't  | Você não era       |
| He wasn't    | Ele não era        |
| She wasn't   | Ela não era        |
| It wasn't    | Ele/ela não era    |
| We weren't   | Nós não éramos     |
| You weren't  | Vocês não eram     |
| They weren't | Eles/elas não eram |

| Interrogativa | be (simple past) |
| ------------- | ---------------- |
| Was I?        | Eu era?          |
| Were you?     | Você era?        |
| Was he?       | Ele era?         |
| Was she?      | Ela era?         |
| Was it?       | Ele/ela era?     |
| Were we?      | Nós éramos?      |
| Were you?     | Vocês eram?      |
| Were they?    | Eles/elas eram?  |
