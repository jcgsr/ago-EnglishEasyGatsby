¬ß Regra Geral: Basta adicionar **s** ao substantivo.

| Singular | Plural      |
| -------- | ----------- |
| guitar   | guitar**s** |
| violin   | violin**s** |

<hr />

¬ß Casos especiais:

a) e o substantivo terminar com **s, ss, sh, ch, x, o**: deve-se adicionar **es**

| Singular   | Plural       |
| ---------- | ------------ |
| ga**s**    | gas**es**    |
| cla**ss**  | class**es**  |
| wi**sh**   | wish**es**   |
| wat**ch**  | watch**es**  |
| bo**x**    | box**es**    |
| potat**o** | potato**es** |

b) Se o substantivo terminar com _consoante_ + **y**, retira-se o **y** e se coloca **ies**.

| Singular    | Plural        |
| ----------- | ------------- |
| lad**y**    | lad**ies**    |
| countr**y** | countr**ies** |

c) Se o substantivo terminar com **f** ou **fe**, retira-se **f/fe** e se coloca **ves**.

| Singular | Plural     |
| -------- | ---------- |
| wol**f** | wol**ves** |

d) Irregulares (mais usados)

| Singular | Plural     |
| -------- | ---------- |
| man      | m*e*n      |
| woman    | wom*e*n    |
| child    | child*ren* |
| person   | _people_   |
| foot     | f*ee*t     |
| tooth    | t*ee*th    |

e) Nunca mudam no plural (mais usados)

1 - sheep

2 - fish

3- deer

4 - swine

5 - buffalo

6 - shrimp

7 - aircraft

8 - spacecraft
