Um dos verbos mais importantes em inglês, o verbo **"to be"** (_ser/estar_) é o mais usado nessa língua. Ele serve para _descrever_ **algo** ou **alguém**.

Pode ser usado para dizer coisas diferentes de acordo com o contexto. Possui três flexões no Presente Simples: **am, is, are**. Confere os seus usos nas tabelas abaixo. Dica: **am** sempre com _I_, **is** sempre com _he, she, it_ e **are** com todos os outros.

Presta atenção que onde houver as traduções _sou, é, somos, são_, pode-se perfeitamente trocá-las por **estou, está, estamos, estão** respectivamente. Outra coisa: na negativa, pode ser tanto **is not, are not** quanto **isn't, aren't**. Porém, "**I am not**" não contrai! Confere as tabelas abaixo para a conjugação completa.

## Conjugação

| Afirmativa | be - ser/estar |
| ---------- | -------------- |
| I am       | Eu sou         |
| You are    | Você é         |
| He is      | Ele é          |
| She is     | Ela é          |
| It is      | Ele/ela é      |
| We are     | Nós somos      |
| You are    | Vocês são      |
| They are   | Eles/elas são  |

| Afirmativa (contraída) | be - ser/estar |

| They're                | Eles/elas são  |
