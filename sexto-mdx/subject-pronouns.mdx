variant: "sexto"

Os **Pronomes Pessoais** são os sujeitos dos verbos. O Pronome pessoal _I_ deve ser sempre escrito com letra maiúscula.

| Subject | Translation |
| ------- | ----------- |
| I       | Eu          |
| You     | Você        |
| He      | Ele         |
| She     | Ela         |
| It      | Ele/Ela     |
| We      | Nós         |
| You     | Vocês       |
| They    | Eles/Elas   |
