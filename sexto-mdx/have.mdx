Um dos verbos mais importantes em inglês, o verbo **"to have"**, é um dos mais usados nessa língua. Ele serve para indicar **posse**. É um verbo irregular. Pode ser utilizado como verbo auxiliar para formar o "Presente Perfeito".

Possui duas flexões no Presente Simples: **have, has**. Confere os seus usos nas tabelas abaixo. Dica: na _afirmativa_ **has** sempre com _he, she, it_ e **have** com todos os outros.

## Conjugação

| ---------- | ------------- |
| I have     | Eu tenho      |
| You have   | Você tem      |
| He has     | Ele tem       |
| She has    | Ela tem       |
| It has     | Ele/ela tem   |
| We have    | Nós temos     |
| You have   | Vocês têm     |
| They have  | Eles/elas têm |
